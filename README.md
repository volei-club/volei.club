# volei.club

Platforma completă pentru managementul clubului tău de volei.

## Despre Proiect

**volei.club** este o aplicație web dezvoltată în Laravel, menită să simplifice administrarea, planificarea antrenamentelor, evidența prezențelor, facturarea abonamentelor lunare și comunicarea cu părinții și jucătorii.

Aplicația este construită special pentru a răspunde nevoilor specifice ale cluburilor de volei, permițând o digitalizare rapidă și eficientă a proceselor de management sportiv.

### Funcționalități Principale

- **Managementul Structurii Clubului:** Gestiunea loturilor de jucători, antrenorilor și grupelor / categoriilor de vârstă.
- **Prezență Digitală:** Monitorizarea prezențelor la antrenamente direct de pe telefon sau desktop.
- **Abonamente & Plăți:** Urmărirea plății abonamentelor lunare și emiterea alertelor / remindere pentru restanțieri.
- **Planificare & Calendar:** Organizarea tabelară a meciurilor (oficiale / amicale) și a antrenamentelor.
- **Statistici & Performanță:** Dashboards personalizate pentru a analiza eficiența și istoricul sportivilor.
- **Portal pentru Părinți:** Interfață dedicată părinților pentru a accesa și monitoriza statusul copilului.

## Tehnologii Folosite

- **Framework:** Laravel 12.x
- **Limbaj:** PHP 8.2+
- **Frontend:** Laravel Blade, Tailwind CSS
- **Bază de date:** MySQL

## Stadiul Proiectului

Aplicația se află momentan în stadiu de dezvoltare (**WIP - Work in Progress**). Toate funcționalitățile de mai sus sunt implementate și rafinate iterativ.
